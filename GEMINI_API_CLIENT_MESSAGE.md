**Subject: Gemini AI API Key Required for League Mapping Feature**

Hello,

We need a **Google Gemini AI API key** for our betting application. This API is used for the league mapping feature where if standard matching methods fail, AI automatically matches leagues.

**Details:**

1. **API Service:** Google Gemini AI
2. **Model Name:** `gemini-2.5-flash`
3. **Purpose:** League mapping fallback (when standard matching fails)
4. **Purchase/Get API Key Link:** https://aistudio.google.com/apikey
   - Alternative: https://makersuite.google.com/app/apikey

**Steps to Get API Key:**

1. Visit Google AI Studio: https://aistudio.google.com/apikey
2. Login with your Google account
3. Click "Create API Key" button
4. Copy the API key
5. Add to environment variables: `GEMINI_API_KEY=your_api_key_here`

**Note:** 
- Free tier is available (limited requests per minute)
- Paid plan recommended for production
- Store API key securely (in environment variables)

**Model Details:**
- Model: `gemini-2.5-flash`
- Fast response time
- Good for matching tasks
- Cost-effective

If you have any questions, please let me know.

Thank you!
